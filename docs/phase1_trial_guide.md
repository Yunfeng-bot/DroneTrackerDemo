# Phase 1 试用说明

## 安装

1. 在手机上安装 `app-debug.apk`。
2. 首次启动时允许相机权限。
3. 如果你要从相册选目标照片，再允许图片读取权限。

## 两种使用方式

1. 方式 A：选目标照片
说明：点击 `选目标照片`，选择一张清晰的目标图片。

2. 方式 B：直接圈住目标
说明：在实时画面上长按并拖框圈住目标。

3. 使用规则
说明：这两种方式任选一种，不需要同时做。

## 你会看到什么

1. 锁定后，红框会跟着目标移动。
2. 目标完全离开画面时，红框会先松开，不应长时间锁住别的物体。
3. 目标重新回到画面后，系统会尝试重新锁定。
4. 如果锁错、跟丢或要换目标，点击 `重置` 后重新操作。

## 导出运行日志

1. 试用结束后点击界面左上角的 `导出日志`。
2. 应用会分享最新一份 `phase1_*.log`。
3. 把该日志文件发给开发者即可。

## 目标照片建议

1. 照片尽量清晰。
2. 尽量选择有细节、有纹理的目标。
3. 目标方向尽量接近当前画面里看到的方向。
